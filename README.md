# The Courage to Know
### A 6-Week Workbook for Emotional Awareness + Motherhood

---

## About This Workbook

*The Courage to Know* is a self-guided workbook designed for mothers who want to understand themselves more deeply — their emotional patterns, their reactions, and the roots of how they show up in relationship with their children.

Using astrology as a lens for self-reflection, this workbook moves through six weeks of guided teaching, personal reflection, and journaling. It is not about becoming a perfect mother. It is about becoming a more aware one.

> *"Your chart is not here to define you. It's a map of patterns. And once you begin to understand those patterns… you begin to create choice."*

## How to Use

This workbook is designed to be used alongside *The Courage to Know* course.

**To view the workbook:**
Visit the live workbook at the link below, or open `index.html` in any web browser.

**To print:**
1. Open `index.html` in Chrome or Safari
2. File → Print (or `Cmd+P` / `Ctrl+P`)
3. Set paper size to **8.5 × 11 in**
4. Set margins to **None**
5. Enable **Background graphics**
6. Save as PDF or print directly

---

## Design

The workbook is built as a single HTML file with no external dependencies beyond Google Fonts. The design follows a luxury journal aesthetic — black, warm gold, and cream — with elegant serif typography, generous white space, and intentional pacing to support emotional reflection.

- **Colors:** Soft black · Warm gold · Cream / Ivory
- **Typography:** Cormorant Garamond (headers) · Jost (body)
- **Format:** 8.5 × 11 in, print-ready
